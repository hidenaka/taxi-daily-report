# Step 1
## GitHub アカウントを作る

1. ブラウザで `github.com/signup` を開く
2. **メールアドレス** を入力 → 続ける
3. **パスワード** を作成 → 続ける
4. **ユーザー名** (英数字、半角) → 続ける
5. パズル認証 → 完了
6. 受信メールの **確認コード** を入力
7. **作ったユーザー名を管理者に LINE で伝える**

> 管理者があなたを「タクシー日報のデータリポジトリ」に招待します
> しばらくすると招待メールが届きます

---

# Step 2
## GitHub アクセスキー (PAT) を発行

1. **招待メール** を開いて **Accept invitation**
2. 右上アイコン → **Settings** → **Developer settings**
3. **Fine-grained tokens** → **Generate new token**
4. Name `taxi-app` / Expiration `1 year`
5. Repository: `taxi-daily-report-data` のみ選択
6. Permissions: Contents = **Read and write**
7. **Generate token** → `ghp_...` を **すぐコピー** (再表示不可)

> 紛失したら同じ手順で作り直せます

---

# Step 3
## アプリをホーム画面に追加

1. 管理者から受け取った **アプリ URL** を開く
2. iPhone (Safari): 画面下中央の **共有ボタン** (□に↑) をタップ
3. メニューを下にスクロール → **「ホーム画面に追加」** をタップ
4. **「追加」** をタップ

> Android (Chrome) の場合: 右上 ⋮ → **「ホーム画面に追加」**
> 普通のアプリのようにホーム画面のアイコンから開けます

---

# Step 4
## 設定画面で 3 つ入力

1. ホーム画面のアイコンから **アプリを開く**
2. 下のメニューから **⚙ 設定** を開く
3. 受け取った 3 つを入力:
   - GitHub PAT (`ghp_xxx...`) ← Step 2 でコピーしたもの
   - データリポジトリ (`hidenaka/taxi-daily-report-data`)
   - userId (`user_xxx`) ← 管理者から指定されたもの
4. **「保存」** をタップ (1 回だけ。次回以降は自動)

---

# Step 5
## Gemini Web で写真をテキスト化

1. ブラウザで `gemini.google.com` を開く
2. **Google アカウントでログイン** (Gmail を持っていればそれで OK)
3. 受け取った **プロンプト** を全部コピー → ペースト
4. **日報の写真をアップロード** (複数枚 OK)
5. 送信 → 数十秒待つ
6. 出力テキストを **全選択 → コピー**

> 20-30 枚で新しいチャットに切り替え (精度低下防止) / Gmail なしなら先に Google 登録

---

# Step 6
## アプリにペーストして保存

1. タクシー日報アプリを開く
2. **📝 入力** メニューを開く
3. テキストエリアに **ペースト**
4. **「パース」** ボタンをタップ
5. プレビューで **件数・売上・キャンセル** を目視確認
6. 車種が違っていたら **手動で修正**
7. **「保存」** をタップ

> 「保存しました」と出れば完了
> 複数日まとめて入れる場合は **📚 一括入力** メニューから
