# Step 1
## アプリをホーム画面に追加

1. 管理者から受け取った `URL` を開く
2. 画面下中央の **共有ボタン** (□に↑) をタップ
3. メニューを下にスクロール → **「ホーム画面に追加」** をタップ
4. **「追加」** をタップ

> 普通のアプリのようにホーム画面のアイコンから開けるようになります
> Android (Chrome) の場合: 右上 ⋮ → **「ホーム画面に追加」**

---

# Step 2
## 設定画面でキーと ID を入れる

1. ホーム画面のアイコンから **アプリを開く**
2. 下のメニューから **⚙ 設定** を開く
3. 受け取った 3 つを入力:
   - GitHub PAT (`ghp_xxx...`)
   - データリポジトリ (`hidenaka/taxi-daily-report-data`)
   - userId (`user_xxx`)
4. **「保存」** をタップ (1回だけ。次回以降は自動)

---

# Step 3
## Gemini Web で写真をテキスト化

1. ブラウザで `gemini.google.com` を開く
2. **Google アカウントでログイン** (Gmail で OK)
3. 受け取った **プロンプト** をコピー → ペースト
4. **日報の写真をアップロード** (複数枚 OK)
5. 送信 → 数十秒待つ
6. 出力テキストを **全選択 → コピー**

> 一度に 20-30 枚までが安定 / プロンプトは管理者から配布

---

# Step 4
## アプリにペーストして保存

1. タクシー日報アプリを開く
2. **📝 入力** メニューを開く
3. テキストエリアに **ペースト**
4. **「パース」** ボタンをタップ
5. プレビューで **件数・売上・キャンセル** を目視確認
6. 車種が違っていたら **手動で修正**
7. **「保存」** をタップ

> 「保存しました」と出れば完了
> 複数日まとめて入れる場合は **📚 一括入力** メニューから
