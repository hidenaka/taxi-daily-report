キャビス プライバシーポリシー

中野秀昭(以下「当方」)は、タクシー乗務員向けスマートフォンアプリ「キャビス」(以下「本サービス」)における個人情報・利用者情報の取扱いについて、以下のとおり定めます。

第1条(基本方針)

当方は、個人情報保護法その他の関係法令を遵守し、本サービスで取得する情報を適正に取り扱います。本サービスの無償機能は匿名アカウントでご利用いただけ、氏名・住所等の個人を直接特定する情報の入力を原則として求めません。有料プランへのお申し込みにあたっては、ユーザーIDとパスワードによるアカウント登録をお願いしています(メールアドレスの登録は任意)。本サービスは設計上、設定画面においても氏名・呼称・ニックネーム等の個人を直接特定する情報の入力欄を設けておらず、サーバーに保存もしません。

第2条(取得する情報)

当方は、本サービスの提供にあたり以下の情報を取得します。

  1. アカウント情報:匿名認証ID(Firebase Authentication が自動発行する識別子)、アプリ内ユーザーID(利用者が設定)。有料プランへのお申し込み時には、アカウント登録のためのユーザーID＋パスワード。メールアドレスは利用者が任意で設定された場合のみ取得します。
  2. 乗務データ(日報):日付、売上、営業回数、乗務時間、歩率設定、推定支給額、車両種別等、利用者が入力または写真取り込みで記録した業務記録。
  3. 乗務タイマー情報:勤務開始・休憩・終了の時刻記録。
  4. 位置情報:利用者が「現在地から営業エリアを推定する」機能を明示的に操作したときに取得する、その時点の緯度・経度。常時取得・追跡は行いません。
  5. 決済関連情報:有料プランの契約状態、契約プラン、課金状態。クレジットカード番号等の決済情報は、決済処理サービス Stripe が直接取得・管理し、当方は保持しません。
  6. 写真取り込み(OCR)の画像:利用者が日報の写真を取り込んだ場合、画像は当方の運営するサーバー(Google Cloud Functions)に送信され、サーバー上のメモリで文字認識処理にのみ使用されます。画像そのものを保存・ログ記録することはなく、処理完了とともにメモリから破棄されます。OCRで読み取られたテキスト(売上・時刻・距離等)のみが、利用者の乗務記録としてデータベースに保存されます。
  7. 利用環境・アクセス情報:アクセス解析サービス(Firebase / Google Analytics)を通じて取得される端末種別、ブラウザ、利用状況等の統計情報。

第3条(利用目的)

当方は、取得した情報を以下の目的で利用します。

  1. 本サービス(日報・乗務タイマー・IC判定・需要予測の各機能)の提供・維持
  2. 歩率・推定支給額の試算、乗務記録の集計・表示
  3. 営業サポート機能における匿名化されたベンチマーク統合分析(エリア・時刻のみを匿名で集計し、利用者全体の傾向として表示)。利用者は設定により本集計への参加をいつでもOFFにできます。OFFにすると本人のデータが集計から除外され、同時に営業サポートの「全員データ統合」機能も利用できなくなります。なお、参加期間中に他の利用者が閲覧した匿名集計(エリア・時刻)は遡って取り消すことができません。
  4. 需要予測機能の精度向上(個人を識別できない形に統計化・匿名化したうえで)
  5. 有料プランの課金・契約管理
  6. 不具合対応、本サービスの改善、利用状況の分析
  7. 利用者からのお問い合わせへの対応、重要なお知らせの連絡

第4条(第三者提供の制限)

  1. 当方は、利用者の同意なく、個人を識別できる情報を第三者に提供しません。
  2. 当方は、利用者個人を識別できる形の乗務データ・労働時間・収入情報を、タクシー会社・労働組合その他の第三者に提供しません。 提携組合・タクシー会社が、本サービスを通じて個々の利用者のデータを閲覧することはできません。提携組合は本サービスを任意の紹介プログラムとして案内するのみであり、利用者の名簿・データを保有しません。
  3. 次の場合は例外とします。
    • 法令に基づく開示請求があった場合
    • 人の生命・身体・財産の保護に必要で、本人の同意取得が困難な場合

第5条(業務委託先・外部サービス)

当方は、本サービスの提供のため、以下の外部サービスを利用します。これらは利用目的の達成に必要な範囲で情報を取り扱います。

画像・写真の取扱い

  1. 「写真から取り込み」(自社OCR)：日報の画像はメモリ上でのみ処理し、ディスクへの保存もログへの出力も行いません。OCR処理が完了すれば画像はメモリから破棄され、外部AIサービスに送信されることはありません。
  2. 外部AI(Google Gemini等)を利用した取り込み：本サービスは外部AIサービスにご自身で画像をアップロードし、出力テキストを本サービスに貼り付けるための支援機能(プロンプト例の提供等)を含みます。当該外部送信は利用者ご自身の判断と責任で行うものであり、外部AIサービスでの画像・テキストの取扱いについては当方の管理外です。詳しくは利用規約第8条をご確認ください。

第6条(情報の保管期間)

利用者の乗務データ・アカウント情報は、利用者が削除操作を行うまで、または最終利用日から12か月を経過するまで保管します。位置情報は営業エリア推定の処理に用いた後、エリア推定結果以外の生の座標を継続保存しません。

第7条(安全管理措置)

  1. 当方は、アクセス制御(Firebase Security Rules)により、利用者が自己のデータにのみアクセスできるよう構成しています。
  2. 通信は SSL/TLS により暗号化します。
  3. 当方は、本サービスの運営に必要な範囲でのみ保存データへアクセスします。
  4. 写真取り込み(OCR)機能で当方サーバーに送信された画像は、サーバー上のメモリで処理にのみ使用され、保存・ログ記録は行いません(処理完了とともにメモリから破棄)。
  5. 当方は、利用者の所属会社名・車両番号・乗務員氏名・呼称・ニックネーム等、所属または個人を直接特定できる情報を、当方のサーバーに保存しない設計を採用しています。会社の区別は内部識別子(slug)のみで行い、保存データだけからは「どこの会社のどの運転手か」を特定できないようにしています。

第8条(利用者の権利)

利用者は、自己の情報について、開示・訂正・削除・利用停止を求めることができます。アプリ内の操作で行えるもの(データ削除等)はアプリから、その他は cabis@taxicabis.com までご連絡ください。本人確認のうえ、法令に従い遅滞なく対応します。

第9条(Cookie・類似技術)

本サービスは、ログイン状態の維持・設定の保存のため、端末のローカルストレージおよび Cookie 等の技術を利用します。アクセス解析のため Google Analytics の Cookie を利用する場合があります。ブラウザ設定によりこれらを無効化できますが、一部機能が利用できなくなることがあります。

第10条(位置情報の取扱い)

本サービスは、利用者が「現在地から営業エリアを推定」する操作を行ったときにのみ、ブラウザの位置情報許可を求め、その時点の座標を取得します。位置情報の常時取得・移動追跡は行いません。位置情報の利用を許可しなくても、エリア推定以外の機能はご利用いただけます。

第11条(本ポリシーの変更)

当方は、必要に応じて本ポリシーを変更します。重要な変更を行う場合は、適用開始日の30日前にアプリ内通知等で周知します。
