# Gemini に投げるプロンプト(写真→テキスト変換)

これは知人(新規ユーザー)が自分の日報写真を Gemini Web に投げてテキスト化するときに使うプロンプトです。**このプロンプトの出力は、PWA の入力画面にそのままコピペできる形式**になっています。

> **管理者が一括取込みする場合**: 同じプロンプトを Gemini Web に投げて出力を `data/paste-here.txt` にペースト → `node scripts/admin-bulk-push.mjs` で push。詳細は `admin-runbook.md` 参照。

## 使い方

1. **Gemini Web を開く**: https://gemini.google.com (Google アカウントでログイン、無料)
2. **下の「プロンプト本文」をすべてコピー**
3. Gemini にプロンプトを貼り付け
4. **日報の写真をすべてアップロード**(複数枚OK、何日分でもOK)
5. 送信
6. Gemini が返した **テキストを全選択 → コピー**
7. **タクシー日報アプリ**:
   - 1日分なら → 入力画面 → テキストエリアにペースト → パース → 保存
   - 複数日分なら → 一括入力ページにペースト → 全部push

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| 写真が読めないと言われる | 解像度を上げて再撮影、または1枚ずつ送る |
| 金額がずれる | カンマや桁を AI が見落としやすい。アプリの「パース」プレビューで目視確認 |
| 日付が空欄 | 日付欄の写りを確認 |
| 出力にコードブロック(```) が付く | プロンプトの「コードブロック不要」を強調して再送信 |
| 同じ日が2回出る | 「1画像=1日として独立処理」を強調して再送信 |
| 車種が「個タク」「営業所」など意図しない値 | 「premium または regular のみ」と注記して再送信 |

## アプリ側で起こる自動処理

ペーストして「パース」を押すと、以下が自動で走ります:

- **キャンセル判定**: 「キ」行/¥400/¥500+0km/¥1000+0km は売上¥0扱い
- **車種が空欄なら**: 入力画面では手動で premium/regular を選択。一括入力では迎車比率で自動推論(「ア」が混じっていれば regular 固定)
- **時刻またぎ**: 帰庫時刻が出庫時刻より早ければ翌日扱い
