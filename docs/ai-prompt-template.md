# AI日報変換プロンプト(知人本人がスマホで使う場合)

このドキュメントは、知人が自分で Gemini / Claude を使って日報写真をテキスト化する場合のプロンプトです。

> **運営者(管理者)が代わりに OCR する場合**は `scripts/import-friend-report.mjs` を使ってください。このプロンプトは知人本人が手動で AI に投げる場合のものです。

## 使い方

1. Gemini または Claude のチャットを開く
   - Gemini Free: https://gemini.google.com
   - Claude Free: https://claude.ai
2. 下の「プロンプト本文」をコピー
3. 日報の写真をアップロード + プロンプトを送信
4. 出力されたテキストを全選択コピー
5. タクシー日報アプリ → 「入力」画面 → テキストエリアに貼り付け → 「パース」 → 内容を確認 → 「保存」

## 推奨 AI

| AI | 強み | URL |
|---|---|---|
| Gemini Free | 画像認識(特に日本語)が強い | gemini.google.com |
| Claude Free | 構造化と指示への忠実さが高い | claude.ai |

どちらも無料枠で十分動作します。読み取り精度が低い場合は両方試してください。

## プロンプト本文(ここから下をコピー)

このタクシー日報の写真を以下のテキスト形式に変換してください。コードブロックや前置きは一切不要、テキストのみを出力してください。

先頭に4行ヘッダー:

```
日付: YYYY-MM-DD
車種: premium または regular
出庫: HH:MM
帰庫: HH:MM
---
```

その後にCSV表(日報のフォーマットそのまま):

```
No,乗車,降車,時間,迎,乗車地,降車地,営Km,男,女,合計
1,07:17,07:38,0:21,迎,大田区上池台4,港区港南2,6.7,1,,"3,600"
休,10:47,11:36,0:49,,江東区青海2,,,,,
2,11:40,11:55,0:15,迎,江東区青海2,中央区銀座8,4.2,2,1,"2,100"
```

ルール:

- **休憩行**: No 列を `休` と書く。乗車時刻=休憩開始、降車時刻=休憩終了。乗車地に休憩場所。それ以外は空欄
- **キャンセル行**: No 列を `キ` と書く。営Km は `0`
- **金額**: 必ずダブルクォートで囲む(カンマ有無に関わらず)。例: `"500"`, `"1,200"`, `"3,600"`
- **¥マーク不要**(数字だけ)、和暦不要(西暦のみ)、NULL や「不明」と書かない
- **迎車**: 「迎」列に `迎`、流しは空欄
- **読み取れない値**: 空欄で構わない。推測しない。日付・車種が不明なら空欄のまま
- **男女列**: 人数(数字)。読み取れなければ空欄
- **営Km**: 実車距離(小数 OK)
- **写真が複数枚にわたる**: すべて同じ日のものとして 1 つの出力にまとめる(時刻順)

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| 写真が読めないと言われる | 解像度を上げて再撮影、または1枚ずつ送る |
| 金額がずれる | カンマや桁を AI が見落としやすい。プレビューで目視確認して修正 |
| 日付が空欄 | 日付欄の写りを確認、複数日分が混ざっていないか確認 |
| 出力にコードブロック(```)が付く | プロンプトの「コードブロック不要」を強調して再送信 |
