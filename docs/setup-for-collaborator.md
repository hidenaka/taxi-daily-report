# セットアップ手順書(知人向け)

このアプリは、タクシー乗務日報を信頼できる知人と共有するためのものです。

## 必要なもの

- スマートフォン(iPhone / Android)
- (オプション) Gemini か Claude の無料アカウント — 自分で写真を変換する場合
- 主催者(運営者)から受け取る:
  - **アプリの URL**
  - **共有用 GitHub Personal Access Token** (PAT)
  - **データリポジトリ名** (例: `username/taxi-daily-report-data`)
  - **あなた専用のユーザー ID** (例: `user_a`)

## 2つの運用パターン

### A. 自分で日報を入れる(推奨・継続的)

1〜5 のセットアップ後、自分で日報写真を AI 変換 → アプリに貼り付け → 保存します。

### B. 主催者に写真を渡す(お試し・低頻度)

セットアップだけ済ませて、写真を主催者に送る。主催者がスクリプトで一括取込します。この場合は手順 1〜4 まででOK。

---

## セットアップ手順

### 1. アプリを開いてホーム画面に追加(PWA インストール)

- 主催者から受け取った URL をスマホのブラウザで開く
- **iPhone (Safari)**: 共有ボタン → 「ホーム画面に追加」
- **Android (Chrome)**: メニュー → 「アプリをインストール」 (or 「ホーム画面に追加」)

### 2. 設定画面を開く

- ホーム画面に追加したアプリアイコンをタップ
- 下部メニューの「設定」を開く

### 3. GitHub データリポジトリと PAT を入力

「GitHub(データリポジトリ)」セクション:
- **データリポジトリ**: 主催者から受け取ったリポジトリ名(例: `username/taxi-daily-report-data`)
- **Personal Access Token**: 主催者から受け取ったトークン
- 「保存」をタップ
- 「接続テスト」で `✓` が並べば OK

### 4. ユーザー ID と表示名を入力

「ユーザー」セクション(設定画面):
- **あなたのユーザー ID**: 主催者から指定された ID(例: `user_a`)
  - 英小文字始まり・英小文字 + 数字 + アンダースコアのみ
  - **本名は使わない**(プライバシー保護)
- **表示名(任意)**: 集計画面の表示用(空欄でも可)
- 「保存」をタップ

ここまでで運用 B(主催者が代行)の場合はセットアップ完了です。

### 5. 初回テスト送信(運用 A の場合)

- 当日(または直近)の日報写真を撮る
- Gemini または Claude を開く
- `docs/ai-prompt-template.md` の「プロンプト本文」をコピー & ペースト + 写真をアップロード
- 出力テキストをすべてコピー
- アプリ → 「入力」画面 → テキストエリアに貼り付け
- 「パース」をタップ → プレビューで件数・合計を確認
- 「保存」をタップ
- ホーム画面に当日の乗務が表示されればセットアップ完了

---

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| 「接続テスト」で失敗 | PAT の権限を確認(`repo` スコープ必須)。リポジトリ名のスペルも確認 |
| 「保存」で失敗 | ユーザー ID が間違っていないか設定画面で確認 |
| AI 出力フォーマットがおかしい | `docs/ai-prompt-template.md` を再確認、写真を撮り直す |
| アプリが古いまま | ブラウザキャッシュをクリア、またはアプリを長押しで再読込 |

---

## 注意事項

- **共有 PAT は他人に渡さないでください**(主催者が管理)
- **ユーザー ID は本名にしないでください**(`user_a`, `taro1`, `driver_x` のような匿名 ID)
- 入力した日報データは主催者と他の参加者全員が閲覧できます(個人収入は各自にしか見えませんが、乗車地・降車地・時間・金額は共有されます)
- 困ったら主催者まで連絡してください

---

## 主催者(運営者)向けメモ

- ユーザー追加時は `data/users.json` に `{ "userId": "user_X", "displayName": "...", "active": true }` を追記
- 退会時は同ファイルで `"active": false` に変更(データは残る)
- 知人から写真をまとめて受け取る場合は `scripts/import-friend-report.mjs` で一括取込可能
